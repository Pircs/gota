## Gota 记账
#### 项目预览
[miniaoe.github.io/gota](https://miniaoe.github.io/gota)

基于 Vue2.x 构建的针对移动端设备记账webApp
主要使用 `vuex + localStorage` 来对数据进行增删改查

#### 技术栈
- Vue2.x
- eCharts
- Vuex
- ES6

#### 项目运行
```
yarn install 

yarn serve
```
